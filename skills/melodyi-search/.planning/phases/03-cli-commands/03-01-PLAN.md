<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI → ExecutionStrategy | CLI 传递配置和参数给执行策略 |
| ExecutionStrategy → ComparisonRecorder | 执行策略调用持久化服务 |
| ComparisonRecorder → DatabaseManager | 持久化服务写入数据库 |
| DatabaseManager → SQLite File | 数据库管理器写入文件系统 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | DatabaseManager | mitigate | SQLite PRAGMA foreign_keys = ON 确保数据完整性 |
| T-03-02 | Information Disclosure | ComparisonRecorder | accept | 数据库文件在本地，无网络暴露，低风险 |
| T-03-03 | Denial of Service | ExecutionStrategy | mitigate | thread.join(timeout=10) 防止后台线程阻塞主流程 |
| T-03-04 | Elevation of Privilege | DatabaseManager | accept | 单用户 CLI 工具，无权限管理需求 |

**Note:** 本 Phase 不新增外部服务或 API，安全风险主要在本地文件操作，现有措施已足够。
